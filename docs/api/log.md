### SetLevel

#### 详细描述
loglevel 根据传入的字符串设置日志级别

disable: 禁用所有日志, fatal: 致命错误, error: 错误, warning: 警告, info: 信息, debug: 调试

Example:
```
loglevel("fatal")
```


#### 定义

`SetLevel(i any)`

#### 参数
|参数名|参数类型|参数解释|
|:-----------|:---------- |:-----------|
| i | `any` |   |
