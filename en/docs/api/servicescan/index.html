servicescan.Scan#

扫描服务指纹的核心函数

详细描述#

定义：#

func servicescan.Scan(hosts: string, ports: string, params ...opt) return (resultChannel: chan *fp.MatchResult, r1: error)

参数#

参数名参数类型参数解释
hostsstring想要扫描的主机，支持 , 分割的网段，IP地址，域名等
portsstring想要扫描的端口
params...opt额外扫描参数

返回值#

返回值(顺序)返回值类型返回值解释
resultChannelchan *fp.MatchResult扫描结果的返回点
r1error
